# Todo, Weather App
> 리액트 네이티브로 구현한 투두 앱과 날씨 앱입니다.

## 주요 기능
### Todo
-   Async Storage를 이용해 상단 헤더의 상태와 할 일을 저장해 사용자가 데이터를 저장할 수 있도록 구현했습니다.

### Weather
-   웹 API를 사용해, 사용자의 위치를 받아 해당 위치의 날씨를 조회할 수 있도록 구현했습니다.
